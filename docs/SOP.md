\# Standard Operating Procedures



\## Daily Checks

\- Verify /health endpoint

\- Review /metrics counters

\- Monitor escalation rate



\## Ticket Monitoring

\- Ensure auto-resolved tickets behave correctly

\- Review escalated tickets for patterns



\## AI Monitoring

\- Watch confidence scores

\- Investigate sudden drops in auto-resolution



\## Deployment

\- Restart service after code changes

\- Validate /health and /metrics post-deploy
